## Compilation

This is what IntelliJ runs to run the java artifacts.

```sh
java \
  -javaagent:path/to/idea-rt \
  -Dfile.encoding=UTF-8 \
  -Dsun.stdout.encoding=UTF-8 \
  -Dsun.stderr.encoding=UTF-8 \
  -classpath path/to/artifactsC:path/to/junit-juniper \
  Main
```

## External Libraries
